Aniversário de cadastro
A cada 12 meses de relacionamento com a MekMix
Regra: A cada 12 meses completos desde a data de cadastro do cliente, o sistema gera um alerta de aniversário de relacionamento.

O que o cliente ganha: Um brinde surpresa definido pelo gestor (ex: amostra de produto, desconto especial, brindes personalizados).

Antecedência: O alerta aparece 7 dias antes da data de aniversário para dar tempo de preparar o brinde e entrar em contato.

O que o sistema faz: Monitora a data de cadastro de todos os clientes e gera alerta 7 dias antes do aniversário.
O que o vendedor faz: Entra em contato com o cliente, celebra o aniversário de parceria e entrega o brinde combinado.
Para vendedores: Esse contato é uma excelente oportunidade de fortalecer o relacionamento. Personalize a mensagem — mencione o tempo de parceria e mostre que a MekMix valoriza o cliente.

Programa de indicação
Cliente que indica ganha 10% na próxima compra
Regra: Quando um cliente indica outro e o indicado fecha o primeiro pedido, o cliente que indicou ganha 10% de desconto no seu próximo pedido.

Como registrar: No cadastro de cada cliente há um campo "Indicado por". Ao fechar o primeiro pedido do novo cliente, o sistema gera automaticamente o alerta de desconto para quem indicou.

Validade: O desconto de 10% expira em 60 dias após ser gerado.

O que o sistema faz: Detecta quando o indicado fecha o primeiro pedido e gera o benefício para quem indicou.
O que o vendedor faz: Entra em contato com o cliente que indicou para informar o benefício e aplica o desconto no próximo pedido.
Para vendedores: Sempre pergunte ao cadastrar um novo cliente se ele foi indicado por alguém. Esse dado é essencial para o programa funcionar corretamente.

Cliente com padrão de compra mensal

Objetivo: Antecipar o contato antes do cliente precisar pedir, demonstrando atenção e facilitando a recompra.

Importante sobre preços: A mensagem de contato não deve mencionar preços. Os valores são negociados pelo vendedor durante a conversa.

Sugestão de mensagem: "Oi [Nome]! 👋 Vi que você tem pedido copos todo mês — posso já verificar disponibilidade para você esse mês?"

O que o sistema faz: Gera alerta quando detecta padrão de 2 meses consecutivos.
O que o vendedor faz: Entra em contato pelo WhatsApp, sem mencionar preços, e abre o pedido se houver interesse.
Para vendedores: Nunca mencione preços na mensagem de prospecção recorrente. O objetivo é facilitar — não pressionar. Deixe o cliente guiar a conversa.
